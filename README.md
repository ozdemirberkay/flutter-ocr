# flutter_ocr

Flutter OCR example project

## Getting Started

**Step 1:**

Download or clone this repo by using the link below:

```
https://github.com/ozdemirberkay/flutter-ocr.git
```

**Step 2:**

Get the required dependencies: 

```
flutter pub get 
```

**Step 3:**

Run the app:

```
flutter run 
```
